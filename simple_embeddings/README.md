# Simple Embeddings

## О задаче

Домашнее задание по векторным представлениям слов и предложений. Основная задача — ранжировать похожие вопросы Stack Overflow по эмбеддингам.

## Что сделано

- загружены предобученные word2vec-эмбеддинги Stack Overflow;
- реализовано получение эмбеддинга предложения как среднего по словам;
- посчитаны косинусные расстояния между вопросами;
- реализованы метрики ранжирования Hits@K и DCG@K;
- разобраны базовые идеи Learning to Rank.

## Стек

- Python
- numpy
- gensim
- scikit-learn
- matplotlib
