# Deep Learning School — Semester 2

Репозиторий с домашними заданиями второго семестра Deep Learning School. Основной фокус — NLP, трансформеры, языковые модели, embeddings и RAG.

## Содержание

| Папка | Тема |
| --- | --- |
| [`simple_embeddings`](simple_embeddings/) | Word embeddings и ранжирование похожих вопросов Stack Overflow |
| [`hw_text_classification`](hw_text_classification/) | Классификация новостей AG News с рекуррентными моделями |
| [`hw_language_modeling`](hw_language_modeling/) | GRU-языковая модель на датасете IMDB |
| [`attention_and_transformers`](attention_and_transformers/) | Fine-tuning трансформеров и визуализация attention-карт |
| [`you_are_bot`](you_are_bot/) | Соревнование по определению бота в диалогах |
| [`hf_rag`](hf_rag/) | Туристический RAG с retrieval, reranking и оценкой качества |

## Стек

- Python
- PyTorch
- HuggingFace Transformers
- Sentence Transformers
