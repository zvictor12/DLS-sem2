# Языковая модель на датасете IMDB

Языковая модель на основе GRU, обученная на датасете IMDB.

## Архитектура

- `nn.Embedding` → `nn.GRU` → `nn.Linear` → `nn.Linear` (проекция на словарь)

- Dropout (0.3), Tanh активация

## Лучший результат

| Модель | Hidden Dim | Эпохи | Перплексия (тест) |
| - | - | - | - |
| GRU (бейзлайн) | 320 | 10 | **67.47** |


## Эксперименты

- **2×GRU (num\_layers=2):** сходимость сильно ухудшилась, перплексия ~199 после 6 эпох

- **RMSProp:** хуже чем Adam, с weight decay и без

- **LayerNorm:** 82.8 перплексии после 10 эпох

- **2 линейных слоя:** плохо; с hidden\_dim=512 — переобучение без улучшения на валидации

- **Hidden dim 256→320:** лучшее улучшение, 67.47 перплексии

## Вывод

Усложнение архитектуры (больше слоёв, больше параметров) не дало выигрыша на этом датасете. Лучший результат — умеренное увеличение размерности скрытого состояния.
