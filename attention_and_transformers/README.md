# Attention and Transformers

## О задаче

Домашнее задание по механизму внимания и классификации математических задач по темам. В качестве backbone используются модели HuggingFace.

## Что сделано

- реализован класс трансформерной модели для классификации;
- добавлена функция заморозки backbone;
- написан цикл дообучения с замороженным и размороженным backbone;
- сравнены `cointegrated/rubert-tiny2` и `tbs17/MathBert`;
- визуализированы attention-карты до и после дообучения;
- сделаны выводы о поведении голов внимания.

## Результаты

Лучше всего показал себя полный fine-tuning MathBert: модель дала примерно на `0.1` F1 выше остальных вариантов.
