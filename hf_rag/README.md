# Tourist RAG

## О задаче

Домашнее задание по построению RAG-системы для туристических и исторических объектов. Система отвечает на вопросы по описаниям достопримечательностей, метаданным и подготовленным текстовым чанкам.

## Что сделано

- проведены EDA и очистка данных с описаниями объектов;
- агрегированы дубли до уровня уникальной достопримечательности;
- построены эмбеддинги через `intfloat/multilingual-e5-base`;
- настроен retrieval в LanceDB;
- добавлен reranking через `BAAI/bge-reranker-v2-m3`;
- выполнена визуализация эмбеддингов и оценка RAG-пайплайна.

## Стек

- Python
- HuggingFace Transformers
- Sentence Transformers
- LanceDB
- LangChain
- RAGAS
- PCA, UMAP
