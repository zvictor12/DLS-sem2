# You Are Bot

## О задаче

Решение соревнования по определению бота в диалогах. Одна строка обучающей таблицы соответствует участнику диалога, а качество оценивается через LogLoss.

## Что сделано

- собрана таблица признаков по участникам диалогов;
- подготовлены варианты текстового контекста: собственные реплики, реплики собеседника и полный диалог;
- добавлены простые числовые признаки текста;
- обучены TF-IDF + Logistic Regression модели;
- протестированы KNN, clustering features, BERT embeddings и fine-tuning;
- подготовлен финальный ensemble top-3 TF-IDF моделей.

## Результаты

Лучшим оказался простой TF-IDF подход по собственным репликам участника. Финальный ансамбль top-3 TF-IDF моделей дал public score около `0.317`.
